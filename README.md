# Personalization
### 資料夾路徑
預設資料夾是放在$HOME底下  
可以編輯note 裡的 dir 變數修改  

### 資料夾名稱
預設資料夾名稱是myUnixNote  
可以編輯note 裡的 note變數修改  

### 加入環境變數
參考 sh ./note bashScript 中關於$PATH的用法  
將資料夾路徑加入path中     
以後隨時都可以用note 系列指令
